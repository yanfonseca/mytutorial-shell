# Python

## Pip
* Verificar se está instalado

python3 -m pip --version

* Se não estiver

sudo apt update
sudo apt install python3-pip
pip3 --version

## Environment

python3 -m pip install --user -U virtualenv
python3 -m virtualenv filepath

* Para ativar a env 
  filepath/bin/activate

* Para desativar a env
  deactivate
